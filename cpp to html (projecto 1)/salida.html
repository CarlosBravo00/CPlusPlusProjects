Programa: Factorial

Descripción: Programa que tiene una función para calcular el factorial de un número
Autor: Juan Perez
Fecha de elaboración 28 de Enero de 2017


Función:factorial

Descripción: Calcula el factorial de un número
Parámetro: iNum valor para el que se va a calcular el factorial. Debe ser entero positivo.
Valor de retorno: factorial del valor iNum
