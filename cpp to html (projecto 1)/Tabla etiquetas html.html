Tabla con etiquetas HTML

Etiqueta Función
< !DOCTYPE html > Es una instrucción para el navegador web sobre en qué versión de HTML está escrita la página.
< html > Le dice al navegador que este es un documento HTML, indica el inicio del documento
< /html > Indica el final del documento
< head > Elemento contiene información meta sobre el documento.(incio)
< /head > Termina el elemento que contiene información meta
< title > Elemento especifica el inicio del título para el documento.
< /title > Elemento especifica el final del título para el documento.
< body > El elemento inicia el contenido de la página visible.
< /body > El elemento termina el contenido de la página visible.
< h2 and h3 > Tipos de títulos , H2 es más grande que H3
< /h2 and /h3 > termina los tipos de títulos
< strong > Pone el texto continuo en negrillas
< /strong > determina el final de < strong >
< br > Inserta un salto de línea.
