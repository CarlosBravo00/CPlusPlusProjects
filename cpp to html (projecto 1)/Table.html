Tabla con etiquetas HTML

Etiqueta Función
< !DOCTYPE html > Es una instrucción para el navegador web sobre en qué versión de HTML está escrita la página.
< html > Le dice al navegador que este es un documento HTML.
< head > Elemento contiene información meta sobre el documento.
< title > Elemento especifica un título para el documento.
< body > El elemento contiene el contenido de la página visible.
< h2 and h3 > Tipos de tamaños para la letra , H2 es más grande que H3
< strong > Pone el texto en negrillas
< br > Inserta un salto de línea.
